# opencoo

Kompilierte Wissensinfrastruktur für KI-Agenten. Nimmt Dokumente und Besprechungsprotokolle in ein Markdown-Wiki pro Domain in Gitea auf; stellt dieses Wiki Agenten über MCP zur Verfügung. Bei der Einnahme vorsynthetisiert, zur Abfragezeit nicht abgerufen — keine RAG, keine Vektor-Datenbank.

> Compiled-knowledge infrastructure for AI agents. Ingests documents and meeting transcripts into a per-domain Markdown wiki in Gitea; serves that wiki to agents via MCP. Pre-synthesized at ingestion, not retrieved at query time — no RAG, no vectors.

This repository is in a pre-publication stage. More information will be available once the first release is tagged.
